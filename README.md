# 🤟 Manos Que Hablan V1.0

### Sistema de Traducción Bidireccional de Lengua de Señas Colombiana (LSC) mediante IA

---

## 📝 Descripción del Proyecto
**Manos Que Hablan** es una solución tecnológica desarrollada para reducir las barreras de comunicación entre la población sorda y los servidores públicos en la ciudad de Neiva, Huila. Utilizando algoritmos de **Visión Artificial** y **Deep Learning**, el sistema traduce en tiempo real la LSC a texto/voz y viceversa, facilitando el acceso equitativo a servicios de salud y trámites administrativos.

## 🚀 Stack Tecnológico
* **Frontend:** React + TypeScript + Three.js (Avatar 3D).
* **Backend:** FastAPI (Python) + WebSockets.
* **IA & Visión:** MediaPipe (Hand/Holistic Tracking) + TensorFlow.js.
* **NLP:** SpaCy (Procesamiento de Lenguaje Natural).
* **Base de Datos:** PostgreSQL.
